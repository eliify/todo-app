# Todo App

Bu proje, ReactJS ve Vite kullanılarak geliştirilmiş bir Todo List uygulamasıdır. Bootstrap 5 ile tasarlanmış olup CRUD işlemleri (ekleme, listeleme, güncelleme, silme) desteklenir. Veriler tarayıcının LocalStorage'ında saklanır.

## Kurulum ve Çalıştırma

### Ön Gereksinimler
- Node.js (versiyon 14 veya üzeri)
- npm veya yarn

### Adımlar

1. **Proje bağımlılıklarını yükleyin:**
   ```bash
   npm install
   ```

2. **Geliştirme sunucusunu başlatın:**
   ```bash
   npm run dev
   ```

Artık uygulamanız http://localhost:5173 adresinde çalışıyor olmalı!

## Teknolojiler

- React
- Bootstrap 5
- Vite
